Pipeline architecture
Bronze
Raw API responses dumped to MinIO as JSONL.gz, partitioned by source/date/hour. Immutable. Replayable.
s3://bronze/<source>/<entity>/dt=YYYY-MM-DD/...
Silver
Normalized in Postgres. SCD2 dimensions, partitioned facts. UPSERT on natural key — idempotent replays.
ON CONFLICT DO UPDATE · BRIN(ts) · btree(instrument_id, ts)
Gold
Materialized views for hot dashboard queries. Refreshed concurrently every 5 min by Celery.
